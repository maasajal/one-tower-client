### Features and Packages

- Swiper Slider
- Sweet Alert Toaster

- Login with GitHub
- Mongodb database
- Node.js server
- Methods: ["GET", "POST", "PUT", "DELETE", "PATCH"]
- View details for login user
- Rent request for login user
- Payment option for member user
- JsonWebToken (jwt)
